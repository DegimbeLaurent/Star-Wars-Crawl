BECODE

Le nouvel espoir

C'est une époque de confinement. A bord de vaisseaux Becode opérant à partir de leurs bureaux, les apprentis ont emporté leur première victoire sur le maléfique Empire de turlututu.

Au cours de la bataille, les apprentis ont réussi user de leurs connaissances du html et du css, acquis par le sacrifice de nombreuse heure de travail, pour détruire l'arme absolue de l'Empire : l’Ignorance.

Poursuivie par des sbires sinistres de l'Empire, les apprentis comprirent qu’ils devaient néanmoins rejoindre leur base sur la planète GITHUB afin de partager ces précieuse connaissance ….
